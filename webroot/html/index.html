    • You can export table by copying to your clipboard or downloaing as either CSV or PDF
    • You can also export charts in any of these formats (PNG, JPEG, PDF, SVG)

    Did you know?

    • By clicking on the arrows next to the column headers you can sort the table view in descending or ascending order.
    • You can see the number of projects selected underneath the table

    Did you know?

    • You can search for projects in the table and select them by clicking on the checkbox on the first column of the roll. To compare simply click the compare button below the chart. Note that when you select a row it changes background-color.

    • You can change the chart type by clicking the Select Chart type button. You can then pick another chart type from the dropdown
    • You can also change the chart parameters to by Ministries, Agencies, Departments (MDA), or States or Year.

    Did you know?

    • When you click on the label below the chart you can remove the disply of that parameter from the chart
    • If you hover over points/bars on the chart is displays amounts in figures.
